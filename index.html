Responsive Web Design

Website automatically adjusts to different devices.

Ensures accessibility and readability on any screen size.

CSS Grid Layout

Organizes content into rows and columns.

Works in two dimensions: Rows and Columns for flexible layouts.

Basic Grid Syntax

main { display: grid; grid-template
        -columns: 30% 30% 30%; }
display: grid; activates grid layout.
grid-template-columns defines number and width of columns.

Grid Example Structure

<main><article>...</article><aside>...</aside></main>

Article = main content
Aside = sidebar content
Use row-gap and column-gap to add spacing.

HTML Tags: <main>, <article>, <aside>

Important HTML tags

<main> = main content
<article> = independent content
<aside> = secondary content

Viewport & Meta Tag

Visible area on screen

<meta name="viewport" content="width=device-width, initial-scale=1.0">
width=device-width → match device width
initial-scale=1.0 → initial zoom level

Media Query

CSS rules that change depending on screen size

@media screen and (min-width:600px){ main { display:flex; } }
Logic: and, only, not

Breakpoint

Screen width where layout changes

Example: min-width:600px

Responsive Images

CSS example: .responsive

img { width:100%; height:auto; }
Maintains proportions.
Use max-width:100% to prevent enlargement.

Responsive Video

video { width:100%; height:auto; }
Video scales according to screen size, same as images.

JavaScript Basics

Makes websites interactive

Works with HTML and CSS for dynamic interactivity like clicks, hovers, validations.

Separation of Concerns

Separate HTML, CSS, JS files

index.html = structure
style.css = design
script.js = functionality

Internal vs External JavaScript

Internal: inside HTML

External: in separate .js file → cleaner, reusable code

JS Statements & Objects

Statements are one line of code; objects contain data & methods

Example: 
bankLoan.calcPayments(60);

JS Output Methods

document.write(), console.log(), alert()

Examples: 
console.log("Hello")
alert("Hi")

JS Functions & Operators

Functions perform tasks; Operators perform calculations & comparisons

Assignment: =
Arithmetic: + - * / %
Logical: && || !
Comparison: == === < > <= >=

Variables, Arrays, Control Flow, Loops

JS stores data, repeats code, and manages decisions

Variables: let/var
Arrays: store multiple values
Control Flow: if/else
Loops: for, while, do while
